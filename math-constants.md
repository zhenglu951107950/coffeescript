## 数学常数-Math Constants
### 问题
你需要使用常见的数学常数，比如π或者e。
### 方案
使用 Javascript 的 Math object 来提供通常需要的数学常数。  
```
Math.PI
# => 3.141592653589793

# Note: Capitalization matters! This produces no output, it's undefined.
Math.Pi
# =>

Math.E
# => 2.718281828459045

Math.SQRT2
# => 1.4142135623730951

Math.SQRT1_2
# => 0.7071067811865476

# Natural log of 2. ln(2)
Math.LN2
# => 0.6931471805599453

Math.LN10
# => 2.302585092994046

Math.LOG2E
# => 1.4426950408889634

Math.LOG10E
# => 0.4342944819032518
```
### 讨论
另外一个例子是关于一个数学常数用于真实世界的问题，是有关[弧度和角度]( http://coffeescript-cookbook.github.io/chapters/math/radians-degrees)这部分数学章节。
